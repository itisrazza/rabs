rabs - Razza's Arch Linux Bootstrap

Razza's Arch Linux Bootstrap is an installation script for Arch Linux.

How to Install

Boot into the Arch Linux installation environment (download), connect to the internet and run the following commands:

curl -O https://rabs.razza.io/rabs-install.sh
chmod a+x rabs-install
./rabs-install

The script will then guide you through the steps of installing Linux on your system.
